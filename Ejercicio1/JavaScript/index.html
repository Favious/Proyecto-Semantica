TRIANGULO DE PASCAL

¿Qué es?

Un triángulo de Pascal es una matriz de números matemáticos que se utiliza para encontrar varios otros conjuntos de números, como secuencias de Fibonacci, números de tetaedro y potencias de dos.

¿Cómo funciona?

Se parte de un triángulo de tres números iguales, para genenar el siguiente nivel se pone el mismo numero a los costados y para generar los del medio simplemente va sumando por pares los numeros del nivel anterior, y así sucesivamente.
